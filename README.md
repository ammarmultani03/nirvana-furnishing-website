🛋️ NIRVAANA — Home Furnishing Studio
NIRVAANA is a high-end, responsive web application designed for a luxury home furnishing brand based in Surat, India. It goes beyond a simple storefront by offering a Real-time Sofa Configurator, a "White-Glove" service booking system, and a seamless, interactive shopping experience.

✨ Key Features
Interative Sofa Studio: A custom-built configurator that allows users to adjust seater size, arm style, foam density, and leg finishes with live price calculation and SVG-based visual previews.

Dynamic Product Catalog: Filterable product grid (Sleep, Sofa, Décor) with quick-view modals, color swatch selection, and smooth "reveal" animations.

Smart Cart System: A persistent side-drawer shopping cart with real-time quantity updates, price formatting (INR), and empty-state handling.

"Free Home Visit" Booking: A comprehensive scheduling form for professional measurements and fabric consultations.

Dark Mode Support: A built-in theme engine that toggles between a "Warm Cream" light mode and a "Deep Chocolate" dark mode.

Festive Marketing Tools: Integrated promotional banners and a dynamic marquee for announcements.

Mobile-First Design: Fully responsive navigation with a specialized burger menu and optimized touch targets for mobile shopping.

🛠️ Technical Stack
Frontend: Semantic HTML5, CSS3 (Custom Properties/Variables, Flexbox, Grid).

Logic: Vanilla JavaScript (ES6+).

Images: Dynamic integration with high-resolution Unsplash APIs for realistic product textures.

Typography: Google Fonts (Cormorant Garamond for elegance, DM Sans for readability).

Animations: Intersection Observer API for scroll-triggered animations and CSS Keyframes for micro-interactions.

🎨 Design Philosophy
The UI is inspired by "Quiet Luxury" — utilizing a palette of Terracotta, Sand, and Bark. It focuses on whitespace, elegant serif typography, and subtle grain overlays to evoke the feeling of a premium physical studio.
